## 前言

继11.22日发布了第一版[jQuery UI Portlet](/jquery/2012/11/22/jquery-ui-portlet.html)之后得到了一些网友的反馈，今天把酝酿了几天的想法全部加入到了新版本中。

## 1. 新特性

1. 添加单视图模式，双击标题栏可以进行切换，并且有动画效果
2. 添加内容高度设置，可以为每个portlet设置高度
3. 添加压缩版本的js文件
4. demo页面大调整，用：左上右的layout布局，方便演示
5. 同步API
